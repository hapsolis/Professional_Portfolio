# Day 84 – Tic Tac Toe (Console Game)

A classic two-player **Tic Tac Toe** game built in Python as part of the 100 Days of Python challenge.

## Features

- Playable in the terminal (text-based)
- Two modes:
  - Human vs Human (2 players)
  - Human vs Computer (simple random AI as opponent)
- Clean board display with position numbers for easy input
- Win detection (rows, columns, diagonals)
- Draw detection
- Option to play again
