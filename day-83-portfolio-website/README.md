# Day 83 – Professional Portfolio

A modern, responsive personal portfolio website built with **Flask** and **Bootstrap 5**.

Showcases my work as a PLC & SCADA Engineer — including real industrial automation projects, certifications, and a working contact form with email delivery.

## Features

- Clean left sidebar navigation (mobile toggle)
- Hero section with typed animation and call-to-action buttons
- About section with professional summary and contact info
- Stats & Skills progress bars
- Resume with experience, education and certifications
- Portfolio section with project cards and detail pages
- Certifications section with individual detail pages (dynamic routing)
- Fully functional contact form that sends real emails via Gmail SMTP
- Responsive design with dark theme, AOS animations and smooth scrolling

## Technologies Used

- **Backend**: Flask + Jinja2
- **Frontend**: Bootstrap 5 + Bootstrap Icons
- **Email**: Flask-Mail + Gmail SMTP
- **Animations**: AOS (Animate on Scroll)
- **Other libraries**: Purecounter, Typed.js, Isotope, Swiper, GLightbox
