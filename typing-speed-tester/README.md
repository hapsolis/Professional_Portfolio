# Day 86 – Typing Speed Test

Desktop GUI application built with Tkinter as part of Dr. Angela Yu's 100 Days of Code Python Bootcamp.

## Features
- 60-second typing test
- Starts timer on first keystroke
- Highlights current word in yellow
- Live WPM and accuracy feedback
- Random sports-themed paragraphs from `paragraphs.txt`
- Results popup with stats

## How to run
1. Make sure you have Python 3 and Tkinter installed
2. `python main.py`

Libraries used: tkinter, threading, random

Inspired by: https://typing-speed-test.aoeu.eu/
